# `opennexus setup`

Initialize `~/.opennexus/opennexus.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
opennexus setup
opennexus setup --workspace ~/.opennexus/workspace
```

To run the wizard via setup:

```bash
opennexus setup --wizard
```
