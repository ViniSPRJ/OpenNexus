# `opennexus pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
opennexus pairing list whatsapp
opennexus pairing approve whatsapp <code> --notify
```
