# `opennexus tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
opennexus tui
opennexus tui --url ws://127.0.0.1:18789 --token <token>
opennexus tui --session main --deliver
```
